Скрипт для скачивания постов по тегам с сайтов на движке danbooru и обновления своей коллекции пикч. Все папки получают имена скачиваемых тэгов. Имена файлов являются комбинацией префикса сервера и ID на сайте. Список серверов хранится в файле servers.txt
Скрипт использует в своей работе утилиты sed, grep и wget.
Для быстрого вывода списка префиксов с серверами используйте ss.sh

Опции командной строки:
 -ua [префикс_сервера] - (Update All) Обновление всей коллекции картинок. Путь к коллекции прописывается при первом запуске. Все новые посты скачиваются в подпапку new каждой *тэго-папки. Механизм предусматривает запоминание ID для последующего скачивания ТОЛЬКО НОВЫХ постов.
 -u [префикс_сервера] тэг - (Update) Обновление только определённой тэго-папки.
 -n [префикс_сервера] тэг - (New) Пополнение коллекции новой тэго-папкой и скачивание всех постов данного тега.
 -l [префикс_сервера] id_поста - (Link) Вывод ссылки на страницу поста
 -sn [префикс_сервера] - (Show New) Открывает папки с новыми постами.
 -gp [префикс_сервера] номер_поста - (Get Post) Скачивание поста по ID. 
 -mv [префикс_сервера] - (Move) Перенос всех новых постов из подпапок new в тэго-папки.
 -vk [префикс_сервера] id_поста - Генерация ссылок для постинга в группу в ВК
 --help - вывод этой справки
 
 Во всех тегах префикс сервера указывать не обязательно - в случае его отсутствия, из servers.txt берётся сервер с ключом default_prefix.
 Так же необязательно указывать ключ командной строки - в таком случае по-умолчанию подразумевается ключ -n.
 Примеры:
 sapphire.sh -ua gel - Обновляет все теги с сервера гельбур
 sapphire.sh -ua - Делает тоже самое, если в servers.txt строчка с сервером Gelbooru имеет ключ default_prefix.
 sapphire.sh -n gel tag_name - Скачивает тег tag_name с сервера gelbooru.
 sapphire.sh tag_name - делает тоже самое, при указанном выше условии.
 *Тэго-папка - Папка с названием скаченного тэга.
