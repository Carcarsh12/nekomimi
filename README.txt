Скрипт для скачивания постов по тегам с сайта gelbooru.com и обновления своей коллекции пикч. Все папки получают имена скачиваемых тэгов. Имена файлов совпадают с ID на сайте. 
Опции командной строки:
 -ua - Обновление всей коллекции картинок. Путь к коллекции прописывается при первом запуске. Из обновления исключаются все тэго-папки*, начинающиеся с плюса. Все новые посты скачиваются в подпапку new каждой тэго-папки. Механизм предусматривает запоминание ID для последующего скачивания ТОЛЬКО НОВЫХ постов.
 -u название_тэга - Обновление только определённой тэго-папки.
 -n название_тэга - Пополнение коллекции новой тэго-папкой и скачивание всех постов данного тега.
 -gp номер_поста [тэго-папка] - Скачивание поста по ID. Перемещает его в указанную тего-папку (опционально).
 -mv - Перенос всех новых постов из подпапок new в тэго-папки.
 --help - вывод этой справки
 *Тэго-папка - Папка с названием скаченного тэга.
