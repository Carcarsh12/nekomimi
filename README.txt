Скрипт для скачивания постов по тегам с сайтов на движке danbooru и обновления своей коллекции пикч. Все папки получают имена скачиваемых тэгов. Имена файлов являются комбинацией префикса сервера и ID на сайте. Список серверов хранится в файле servers.txt
Для быстрого вывода списка префиксов с серверами используйте ss.sh

Опции командной строки:
 -ua префикс_сервера - (Update All) Обновление всей коллекции картинок. Путь к коллекции прописывается при первом запуске. Из обновления исключаются все тэго-папки*, начинающиеся с плюса. Все новые посты скачиваются в подпапку new каждой тэго-папки. Механизм предусматривает запоминание ID для последующего скачивания ТОЛЬКО НОВЫХ постов.
 -u префикс_сервера тэг - (Update) Обновление только определённой тэго-папки.
 -n префикс_сервера тэг - (New) Пополнение коллекции новой тэго-папкой и скачивание всех постов данного тега.
 -l префикс_сервера id_поста - (Link) Вывод ссылки на страницу поста
 -sn префикс_сервера - (Show New) Открывает папки с новыми постами.
 -gp префикс_сервера номер_поста - (Get Post) Скачивание поста по ID. 
 -mv префикс_сервера - (Move) Перенос всех новых постов из подпапок new в тэго-папки.
 --help - вывод этой справки
 *Тэго-папка - Папка с названием скаченного тэга.
